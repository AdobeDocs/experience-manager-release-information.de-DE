# AEM [!DNL Forms] Veröffentlichungen {#aem-forms-releases}

Adobe Experience Manager [!DNL Forms] ist eine Anwendung, die auf AEM bereitgestellt wird. Die Lösung erweitert die Erlebnisse für Web und Mobilgeräte auf Formulare und Dokumente für Unternehmen, wodurch Firmen komplizierte digitale Interaktionen durchführen und gleichzeitig die Benutzererfahrung verbessern und ihre Reichweite vergrößern können. Der vordefinierte AEM [!DNL Forms], der im AEM Quick Beginn enthalten ist, verfügt über einen eingeschränkten Funktionsumfang. AEM [!DNL Forms]-Add-On-Paket bietet erweiterte Funktionen wie Integration mit [!DNL Adobe Analytics], [!DNL Adobe Sign], Dokument-Services, Workflow-Engine und vieles mehr.

>[!NOTE]
>
>AEM Forms veröffentlicht die Add-On-Pakete eine Woche nach dem geplanten AEM Service Pack- und Cumulative Fix Pack-Veröffentlichungsdatum.

Die folgende Liste enthält alle AEM [!DNL Forms] Add-On-Pakete, die bis heute veröffentlicht wurden, die entsprechenden AEM (Voraussetzungen), Downloadlinks von Paketen und weitere hilfreiche Informationen:

## Zu beachtende Dinge: {#things-to-remember}

* Die oben aufgeführten AEM Forms Add-On-Pakete gelten für AEM [!DNL Forms] unter OSGi und AEM [!DNL Forms] unter JEE.
* Alle AEM [!DNL Forms]-Add-On-Pakete und Schnellkorrekturen sind kumulativ. d. h. ein Add-On-Paket und die Schnellkorrektur enthalten alle vorherigen Add-ons und Schnellkorrekturen für eine Komponente für die entsprechende Version. Wenn Sie beispielsweise Quick Fix 2.4.50 installieren, enthält es bereits AEM [!DNL Forms] Add-On-Paket 2.4.26.

   >[!NOTE]
   >
   >Kumulative Add-On-Pakete und Schnellkorrekturen implizieren, dass die ersten beiden Zahlen der Version identisch sind.

* Installieren Sie immer das neueste verfügbare Service Pack/Feature Pack-Add-on. Es wird dringend empfohlen, da es Sicherheits-, Leistungs-, Stabilitäts- und wichtige Fehlerbehebungen und Erweiterungen umfasst, die seit der allgemeinen Verfügbarkeit der Basisversion veröffentlicht wurden.
* Wenn Sie ein Service Pack installieren oder ein Upgrade von einer früheren Version von AEM auf eine neuere Version durchführen, installieren Sie immer das entsprechende AEM [!DNL Forms] Add-On-Paket. Wenn Sie beispielsweise von AEM 6.1 [!DNL Forms] Add-On-Paket Version 2.2.4 auf AEM 6.1 Service Pack 1 aktualisieren, installieren Sie nach der Installation von AEM 6.1 Service Pack 1 AEM [!DNL Forms] Add-On-Paket Version 2.4.50.
* Die Deinstallation des AEM [!DNL Forms] Add-On-Pakets wird nicht unterstützt.
